Консольное приложение написанное на языке программирования C++. 
Программа делает следующее:
  1. Просит пользователя ввести целое положительное число.
  2. Выводится сообщение:"Число имеет следующие делители:" 
  3. Выполняется цикл с параметром:
     1. Начальное значение параметра уставливается равным 1, а конечное - числу, введённому с клавиатуры.
     2. Выполняется серия цикла, где вычисляется остаток от деления числа, введённого с клавиатуры на переменную-параметр цикла.
     3. Если остаток равен нулю, то значение параметра цикла выводится на экран.
     4. Значение параметра цикла сравнивается с конечным. Если оно меньше цикл выполняется. В противном случае - завершается.
     5. Переменная параметра цикла увеличивается на шаг цикла. В данном примере он равен 1.    
  4. Программа ждёт нажатия клавиши "ввод" перед завершением. 
Разработано в интегрированной среде Microsoft Visual Studio.
